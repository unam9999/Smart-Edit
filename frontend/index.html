03 — Capabilities

Built for precision.

⊞
AI Classification

EfficientNet-B0 with ImageNet weights. Blur detection via Laplacian variance. Face detection via Haar cascades. Multi-signal pipeline.

◈
Smart Filters

Per-folder preset filters — portrait smoothing, landscape HDR, automotive clarity — applied at the pixel level using OpenCV and PIL.

↓
Instant Export

Processed images delivered as a sorted ZIP archive with category sub-folders and a CSV manifest. Upload → Classify → Download.
